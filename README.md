# Read Me Generator

A simple Web app to help organize your 9-5 day

Bootcamp Challenge 09: NodeJS

Description

This repository was created to record the creation of a read me generator. The primary motivation for this repo is to create a easy application that can be used tocreate read me files. The secondary motivation for this repo was to provide easy access to a clear and completed work flow of the changes made to the code to meet the assignment critieria. The primary problem we aim to solve is automating read mes!

 
![demo](/demo.gif)

Installation

Download the zip file or repository. To start the app, open the terminal and run node index.js.

Usage

User will be prompted with a series of questions to fill in for their Read ME. If it is blank please use N/A instead. Successfully ansering the questions will result in the read me being generated.
